# 🌟 Quiz Website 🌟  

A simple and interactive quiz website where users can answer multiple-choice questions and get their results instantly!  

---

## ✨ Features  

- ✅ **Interactive Quiz Interface**  
- 📝 **Multiple-Choice Questions**  
- 📊 **Real-Time Score Calculation**  
- 🎨 **Responsive and User-Friendly Design**  

---

## 📂 Project Structure  

```
Quiz Website/
├── Code Files/
│   ├── main.html
│   ├── contact.html
│   ├── contact_style.css
│   ├── main_style.css
│   ├── script.js
├── Images
├── README.md
```

- **Code Files**: Contains the core files for the website (HTML, CSS, JS).  
- **Images**: Contains visual assets such as logos, banners, and other images used on the website.  

---

## 🚀 How to Run  

1. Clone or download the project to your local machine.  
2. Navigate to the `Code Files` folder.  
3. Open the `main.html` file in any modern web browser.
